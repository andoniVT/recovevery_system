Una base de datos de red es una base de datos conformada por una coleccion o set de registros, los cuales estan conectados entre si por medio de enlaces en una red. El registro es similar al de una entidad como las empleadas en el modelo relacional.
Un registro es una coleccion o conjunto de campos (atributos), donde cada uno de los que contiene solamente un unico valor almacenado, exclusivamente el enlace es la asociacion entre dos registros, asi que podemos verla como una relacion estrictamente binaria.
Una estructura de base de datos de red, llamada algunas veces estructura de plex, abarca mas que la estructura de arbol, porque un nodo hijo en la estructura red puede tener mas de un nodo padre. En otras palabras, la restriccion de que en un arbol jerarquico cada hijo puede tener solo un padre, se hace menos severa.
Asi, la estructura de arbol se puede considerar como un caso especial de la estructura de red.
